HANDOFF CONTRACT
--------------------------------------------------------------------------------

Module 1 produces:
  {'ecosystem_condition': 0.35, 'regulating_es': 0.45, 'low_pressure': 0.20}

Module 2 expects:
  weights['group_a_weights'] = {
      'ecosystem_condition': float,
      'regulating_es': float,
      'low_pressure': float
  }

Validation ensures:
  ✓ Keys match expected Group A criteria
  ✓ Weights sum to 1.0 ± 1e-6
  ✓ No negative weights
  ✓ Zero-weight criteria excluded
  ✓ Clear error messages

KEY FUNCTIONS
--------------------------------------------------------------------------------

validate_weight_handoff(weight_dict, config_path)
  Validates weight dictionary against handoff contract.
  Raises ValueError with detailed message if contract violated.

format_weights_for_mce(weight_dict)
  Filters zero-weight criteria and normalises to exactly sum = 1.0.
  Returns dict ready for Module 2 MCE engine.

Usage:

      validate_weight_handoff,
      format_weights_for_mce
  )

  # Validate
  validate_weight_handoff(weights, "config/criteria_defaults.yaml")

  # Format
  mce_weights = format_weights_for_mce(weights)
